x좌표 y좌표 최단거리 구하기 
1. 준비물을 준비해준다.
    * 목표좌표 N, M 
    * 미로행렬 
    * 방문여부 [] ->확인 위해 체크 
2. bfs 함수 
    * 파라터는 : x, y좌표 
    * dx = [-1, 0, 1, 0] / dy = [0,1,0,-1]
    * 빈큐 queue = []
    * 시작할때 빈 큐를 채워준다. 첫좌표 0,0으로 
    * 반복문 시작 
    * 목표 : 큐에서 요소 반환 
    * 동서남북 돌아가면서 갈 수 있는지 확인 (반복문 4까지)
    * nx = 목표x좌표+ dx+1
    * ny = 목표y좌표+ dy+1
    * 0이거나, 범위를 벗어난 경우 다음 좌표이동, 1이면, queue에 좌표추가 하고, 방문여부에 추가 
   *  graph[nx][ny] = graph[target.x][target.y] + 1; //한칸씩 추가 
    * 방문여부가 결과값이 됨.
